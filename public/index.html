v.w

vaporwave.world

Web Developer

A meditation on a single service site a la the now defunct Barack Obama Is Your New Bicycle.

The JavaScript cycles through an array of text, picks an element of said array, and prints it in a style fitting within the vaporwave aesthetic. I always found it very irritating that past versions of this concept forced the user to hit 'refresh' in their browser to reload the page and get a new phrase, so I implemented a feature that allows the user to refresh the page by clicking anywhere on the actual webpage.

Take a look: http://vaporwave.world
